在 React 组件外使用 Model

在实际业务场景中，我们可能需要在 React 组件外消费 Model。如需要在工具函数中访问 state、执行 actions 等。这一节我们将演示如何在 React 组件外使用 Model。

创建 store#

在组件外使用 Model 之前先需要创建一个 store，然后可以通过 store 来加载 Model。

const store = createStore();

这样我们就创建了一个 Store，底层使用的是 Redux Store，所以你可以使用它的任何方法如 getState。 同时我们也兼容 Redux 丰富的生态。如中间件，可以在创建 Store 的时候传入中间件参数。

const store = createStore({  middlewares: [someMiddleware]});

使用 store 加载 Model#

上面我们创建了 store，接下来我们可以通过 store.use 函数将 countModel 挂载到 store 上。

const [state, actions, subsribe] = store.use(countModel);
subsribe(() => {  console.log('state changed', store.use(countModel)[0]); // state changed {value: 2}});
console.log(state); // {value: 1}
actions.add();
console.log(store.use(countModel)[0]); // {value: 2}

依次打印：

{value: 1}state changed {value: 2}{value: 2}

store.use 返回了 state、 actions、subsribe：

  • state: 该 model 的状态。
  • actions: 可执行的 actions。
  • susbribe: 用于监听 model 更新的函数。

在计数器应用中使用#

接下来，我们在计数器应用中加上组件外使用 Model 的方式。去掉了手动点击 add 按钮增加计数器的方式，而是在组件外通过一个定时任务每隔一秒钟自增一。

import { useModel, createStore, Provider } from "@modern-js/runtime/model";import countModel from "./models/count";
const store = createStore();
const [, actions] = store.use(countModel);
setInterval(() => {  actions.add();}, 1000)
function Counter() {  const [state, actions] = useModel(countModel);
  return (    <div>      <div>counter: {state.value}</div>    </div>  );}
export default function App() {  return <Provider store={store}>    <Counter />  </Provider>;}
  • 使用 createStore 创建 store。
  • 使用 store.use(countModel) 来挂载 countModel，并获取到 state 和 actions。
  • 使用 Provider 组件包裹 Counter 组件，并传入我们上面创建的 store。
