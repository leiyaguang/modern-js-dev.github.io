useLoader

useLoader 是一个同构的 API，通常会用来做异步请求等。当 SSR 的时候，服务端使用 useLoader 预加载数据，同时客户端也会复用这部分数据。

API#

函数签名
export interface LoaderOptions<Params = any, TData = any, TError extends Error = any> {  /**   * Revoke when loader excuted successfully.   */  onSuccess?: (data: TData) => void;  /**   * Revoke when loader ended with error   */  onError?: (error: TError) => void;  /**   * initialData to display once loader is ready.   */  initialData?: TData;  /**   * wheather skip loader   * if true, the loader will not exec.   */  skip?: boolean;  /**   * User params, it will bypass to loader's second parameter.   */  params: Params;  /**   * wheather loader can exec on build phase.   */  static?: boolean;}declare type LoaderFn<P = any, T = any> = (context: any, params: P) => Promise<T>;declare const useLoader: <TData = any, Params = any, E = any>(loaderFn: LoaderFn<Params, TData>, options?: LoaderOptions<Params, any, any>) => {  reload: (params?: Params | undefined) => Promise<any> | undefined;  loading: boolean;  reloading: boolean;  data: TData;  error: E;};

参数#

  • loaderFn: LoaderFn<Params, TData>，是 Loader 函数，返回 Promise。
  • options: LoaderOptions<Params, any, any> 选项。
    • onSuccess: (data: TData) => void Loader 成功的回掉。
    • onError: (error: TError) => void Loader 失败的回掉。
    • initialData: TData 初始值。
    • skip: boolean 若设置为 true，则跳过当前 Loader 执行。
    • params: Params 额外的参数，根据 params 序列化的字符串作为 Loader 的唯一标识，当 params 序列化的结果改变的时候，Loader 会自动执行更新。params 也会作为 LoaderFn 的第二个参数。
    • static: boolean，若设置为 true，useLoader 用于 SSG 编译阶段静态数据的获取。

返回值#

  • data: TData Loader 函数成功时返回的数据。
  • loading: boolean 是否处于初始化 loading 状态。
  • error: E Loader 函数 rejected 时的错误信息。
  • reload: (params?: Params) => Promise<any> | undefined 执行该函数可以强行重新执行 loader。
    • params?: Params 当 params 为 undefined，Loader 执行时将复用上次的 params。当 params 不为空，则会使用新的 params 参数重新执行 Loader。
  • reloading: boolean 执行 reload 后，则 reloading 状态为 true。

示例#

普通使用#

function Container() {  const { data, error, loading } = useLoader(    async (context, params) => {      console.log(params) // nicole      return fetch(user);    },    {      onSuccess: data => {        console.log('I did success:(',  data);      },      onError: error => {        console.log('I met error:)',  error);      },      initialData: { name: 'nicole', gender: 'female' },      params: 'nicole'    }  );
  return ...;}
