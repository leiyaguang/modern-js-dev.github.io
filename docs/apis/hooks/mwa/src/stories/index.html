src/**/*.stories.[tj]sx

应用项目 Storybook 调试文件。

可以在项目源码目录(src)下创建 *.stories.[tj]sx 格式的文件作为 Storybook 的调试文件。

在项目下执行 dev story 命令，支持使用这些文件在 Storybook 中对相关内容进行调试。

note

使用 Storybook 需要提前在项目下执行 new 命令启用「Visual Testing (Storybook)」模式。
